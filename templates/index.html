Getting Started with Linux

1. Accessing the Terminal

The terminal is the command-line interface in Linux, used to execute commands.

  • Open it by pressing Ctrl + Alt + T (common shortcut).
  • Search for "Terminal" in the application menu.

2. Basic Commands

  • pwd: Print the current working directory. 
    pwd
  • ls: List files and directories in the current directory. 
    ls
  • cd: Change directory. 
    cd /path/to/directory
  • mkdir: Create a new directory. 
    mkdir new_directory_name
  • touch: Create a new empty file. 
    touch filename.txt
  • rm: Remove files or directories. 
    rm filename.txt

3. Editing Files

Use text editors like nano or vim to edit files:

  • Open a file with nano: 
    nano filename.txt
  • Save changes in nano: Press Ctrl + O, then Enter.
  • Exit nano: Press Ctrl + X.

4. Managing Packages

Install or update software using package managers:

  • For Debian-based systems (e.g., Ubuntu): 
    sudo apt update
    sudo apt install package_name
  • For Red Hat-based systems (e.g., Fedora): 
    sudo dnf install package_name

5. Getting Help

  • Use man to read manual pages for commands: 
    man command_name
  • Use --help to see usage instructions: 
    command_name --help
